# Bingo Game 75 with 6-Card System 🎲🗂️

A feature-rich **75-Ball Bingo** application built with **Flutter**. This app provides a complete bingo experience, featuring a professional **6-card simultaneous display** and an intelligent **auto-check system** that marks drawn numbers in real-time.

[![Google Play](https://img.shields.io/badge/Google_Play-Download-green?logo=googleplay&logoColor=white)](https://play.google.com/store/apps/details?id=jp.aosystem.bingogame)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 Key Features
- **Standard 75-Ball Format:** Optimized for the classic 75-number bingo rules.
- **Integrated 6-Card Display:** Allows players to track and play with six cards simultaneously on a single screen.
- **Real-Time Auto-Check:** Numbers are automatically verified and marked on the cards as soon as they are drawn by the caller.
- **Global Accessibility:** Fully localized in **50+ languages**, supporting bingo fans from all over the world.

## 💡 Best For...
- **Bingo Enthusiasts:** Anyone looking for a professional and automated way to play 75-ball bingo.
- **Solo Gaming:** A stress-free experience where the app handles the tracking of multiple cards for you.
- **Event Showcases:** Demonstrating a modern, digital approach to the traditional bingo game.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Advanced state management to synchronize the 1-75 random generator with multi-card data structures in real-time.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
