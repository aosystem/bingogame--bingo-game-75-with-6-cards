# Bingo Game Machine 🎲

A digital Bingo machine built with **Flutter**, designed for parties, events, and family gatherings. This app replaces the traditional physical bingo cage with a sleek, high-speed digital solution.

[![Google Play](https://img.shields.io/badge/Google_Play-Download-green?logo=googleplay&logoColor=white)](https://play.google.com/store/apps/details?id=jp.aosystem.bingogame)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🌟 Key Features
- **Fair Random Selection:** Advanced logic ensures completely random number generation.
- **History Tracking:** View previously drawn numbers at a glance.
- **Global Accessibility:** Fully localized in **50+ languages**, making it usable for international events.
- **Cross-Platform Ready:** Built with Flutter, showcasing a single codebase that can target multiple platforms.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Efficient state management for tracking drawn numbers and randomizing results.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
